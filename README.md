# 🌍 GeoTab Quiz 1.0 - Chrome Extension

**GeoTab Quiz 1.0** transforms your new tab page into an engaging geography quiz, helping you enhance your knowledge of countries, capitals, flags, and more — one tab at a time! ✨

---

## ✨ Features

- 🧠 **Interactive Quizzes**  
  A fun way to challenge yourself with quick geography questions.

- 🌐 **Topics Galore**  
  Explore questions on countries, capitals, flags, and maps.

- 🎯 **Minimal UI**  
  Sleek and distraction-free layout for focused learning.

- 🔄 **Fresh Question on Every Tab**  
  Open a new tab, get a new quiz — it’s that easy!

---

## 🛠️ Installation

1. **Clone or Download the Repo**:

   ```bash
   git clone https://github.com/AmoghShukla/GeoTab-Quiz-1.0-.git
   ```

   Or download the ZIP from the [Repo](https://github.com/AmoghShukla/GeoTab-Quiz-1.0-) and extract it.

2. **Load the Extension in Chrome**:

   - Open Chrome and go to `chrome://extensions/`
   - Enable **Developer Mode** (top right corner)
   - Click **Load unpacked**
   - Select the extracted folder

You're all set! 🚀

---

## 🚀 How to Use

- Open a new tab — the quiz loads instantly.
- Select an answer from the options.
- Get immediate feedback and learn as you go.
- Open more tabs for more quiz fun! 🎉
